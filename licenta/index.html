Contextul Problemei

Deep Web-ul reprezintă o parte vastă și în mare parte inexploată a internetului, estimată să fie de 400-500 de ori mai mare decât web-ul de suprafață. Această cercetare explorează metodologiile de navigare și analiza conținutului din această zonă obscură a internetului.

Motivația

• Motivația personală a acestei lucrări este de a aduna cât mai multe cunoștințe despre acest domeniu pentru a îmi deschide un orizont nou în ceea ce privește o viitoare posibilă carieră în securitate cibernetică.

• Motivația profesională a acestei lucrări este de a înțelege mai bine riscurile și oportunitățile asociate cu Deep Web-ul, atât din perspectiva utilizatorilor legitimi cât și a celor malițioși. Această cercetare își propune să contribuie la dezvoltarea unor soluții de securitate cibernetică mai eficiente.

Obiective

• Identificarea și catalogarea principalelor metode de acces
• Analiza riscurilor și măsurilor de securitate

Ce este internetul?

Internetul este o rețea globală care permite schimbul de date între miliarde de dispozitive.

Structura internetului

  • Surface Web este partea internetului public accesibilă, prin intermediul motoarelor de căuatre clasice(Google, Bing, Yahoo).
  • Deep Web este reperzentat de conținutul care nu este indexat de către motoarele de căutare. Include date cu acces restricționat(care necesita o parola), dar legal.
  • Dark Web este o parte a Deep Web-ului care este intenționat ascunsă și necesită software specializat pentru acces, cum ar fi Tor. Este adesea asociat cu activități ilegale, dar include și conținut legitim.

Metode de căutare pe Deep Web

• Limitările motoarelor de căutare convenționale: Majoritatea conținutului din Deep Web nu este indexat de motoarele de căutare tradiționale, ceea ce face dificilă găsirea informațiilor relevante.

• Motoare de căutare specializate pentru Deep Web: Există motoare de căutare dedicate, cum ar fi Ahmia sau Not Evil, care indexează conținutul din rețelele de anonimizare.

Metode de publicare a unui site pe Deep Web

• OnionShare este o aplicație open-source care permite partajarea anonimă de fișiere, găzduirea de site-uri web statice sau crearea de chat-uri private direct prin rețeaua Tor, fără a necesita configurarea unui server web tradițional. Prin OnionShare, utilizatorul poate genera rapid un serviciu .onion temporar, accesibil doar celor care dețin link-ul unic generat, asigurând astfel confidențialitatea și securitatea transferului de date pe Deep Web.

• Configurarea manuală a unui server pe Deep Web poate implica utilizarea unor instrumente precum mkp224o pentru generarea de adrese .onion personalizate (vanity). Acest proces presupune rularea unui server web (ex: Apache, Nginx) configurat să accepte conexiuni prin rețeaua Tor, precum și gestionarea atentă a cheilor private și a setărilor de anonimitate pentru a asigura securitatea și confidențialitatea serviciului publicat.

Principalele servicii ale Deep Web

• Forumuri și comunități online: Acestea oferă un spațiu pentru discuții despre subiecte variate, de la tehnologie la activități ilegale, și sunt adesea protejate prin criptare.

• Servicii de mesagerie criptată: Platforme precum Ricochet sau Tor Messenger permit comunicarea anonimă și sigură între utilizatori, protejându-le identitatea și conținutul mesajelor.

• Magazine online (ex: Darknet markets): Acestea sunt platforme de comerț electronic care facilitează tranzacții anonime, adesea cu bunuri și servicii ilegale, dar și cu produse legitime.

• File-sharing: Servicii precum ZeroBin sau Pastebin permit partajarea de fișiere și texte într-un mod anonim, fără a necesita înregistrare sau identificare.

Ce este și la ce se folosește

Distribuția Kodachi, este o distribuție Linux, bazată pe debian, creată de un singur dezvoltator, un expert în securitate din Oman. Distribuția Kodachi este special concepută pentru a naviga în mod anonim, deoarece redirecționeză tot traficul prin VPN+Tor oferind în plus si protecție DNS, firewall, criptare, etc

Tor over VPN vs VPN over Tor

  • Explicație tehnică

    • • Tor over VPN: Traficul este criptat de VPN înainte de a fi trimis prin rețeaua Tor, oferind un strat suplimentar de securitate și anonimitate.

    • VPN over Tor: Traficul este trimis prin rețeaua Tor înainte de a fi criptat de VPN, ceea ce poate oferi un nivel diferit de anonimitate și securitate.


  • Avantaje și dezavantaje

    • • Tor over VPN: Avantajul principal este că ISP-ul nu poate vedea că se folosește Tor, dar VPN-ul poate vedea că se folosește Tor.

    • VPN over Tor: Avantajul este că ISP-ul nu poate vedea nici VPN-ul, dar poate fi mai lent și mai complicat de configurat.


  • Când și cum le folosim

    • • Tor over VPN este recomandat pentru utilizatorii care doresc să ascundă utilizarea Tor de ISP și să beneficieze de viteze mai mari.

    • VPN over Tor este util pentru utilizatorii care doresc să ascundă identitatea VPN-ului, folosit atunci când accesăm un serviciu care restricționează traficul provenit din rețeaua Tor, dar necesită o configurare mai complexă.

Implementarea lor in distribuția Kodachi

• Distribuția Kodachi permite utilizatorilor să configureze ușor Tor over VPN sau VPN over Tor prin intermediul interfeței grafice, oferind opțiuni preconfigurate pentru ambele scenarii, facilitând astfel accesul la Deep Web într-un mod sigur și anonim.

Utilizarea acesteia în contextul Deep Web

• Distribuția Kodachi este ideală pentru navigarea pe Deep Web, deoarece oferă un mediu securizat și anonim, protejând utilizatorii de amenințările cibernetice și asigurând confidențialitatea datelor personale.

• Utilizatorii pot accesa site-uri .onion și alte resurse din Deep Web fără a fi expuși riscurilor de securitate, beneficiind în același timp de viteze de navigare mai mari comparativ cu alte distribuții care nu integrează aceste tehnologii.

Concluzii

• Sinteza principalelor rezultate

• Recomandări de bune practici

Sinteza principalelor rezultate

• Navigarea pe Deep Web necesită instrumente și tehnici specifice pentru a asigura securitatea și anonimitatea utilizatorilor.

• Distribuția Kodachi oferă un mediu sigur și eficient pentru accesarea Deep Web-ului, integrând tehnologii precum Tor over VPN și VPN over Tor.

Recomandări de bune practici

• Folosirea unui VPN încă din faza cercetării domeniului, mai ales în momentul descărcării distribuției Linux sau a platformelor Tor deoarece, chiar dacă sunt open-source acestea sunt supravegheate constant de către organizațiile speciale care monitorizează securitatea cibernetică.

• Utilizarea distribuțiilor specializate, cum ar fi Kodachi, pentru a naviga în mod sigur pe Deep Web.

• Configurarea corectă a rețelelor de anonimizare și a VPN-urilor pentru a proteja identitatea și datele personale.

• Conștientizarea riscurilor asociate cu Deep Web-ul și adoptarea măsurilor de precauție adecvate.

• Participarea la comunități de securitate cibernetică pentru a rămâne la curent cu cele mai bune practici și tehnici de protecție.
